##Problem Statement

Managing student attendance and academic performance manually is time-consuming and error-prone. Teachers often struggle to maintain accurate records, calculate attendance percentages, compute average marks, and assign grades efficiently. This project aims to provide a simple, automated system to streamline these tasks and reduce manual workload.


##Scope of the Project

The project covers essential functions required for small-scale academic record management. It includes adding students, storing attendance, recording marks, generating reports, calculating attendance percentages, calculating average marks, and assigning grades based on predefined criteria. The system runs on the console, making it easy to operate without advanced technical knowledge.



##Target Users

School/college teachers

Coaching institute instructors

Students who want to track their own performance

Small academic groups managing attendance and marks manually



##High-Level Features

Add and store student details

Record daily attendance

Maintain marks for each student over multiple tests

Automatically calculate attendance percentage

Compute average marks

Assign grades based on performance

Generate a detailed student performance report

Save and load data for future use
